Nombre del directorio: python-hello_world
En este directorio se encuentran programas y/o funciones utilizando print() en python
---
Contenidos del directorio:
2-print.py - Imprime "Programming is like building a multilingual puzzle.

3-print_number.py - Imprime una variable int y luego "Battery street".
