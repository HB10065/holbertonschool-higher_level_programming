Nombre del directorio: python-if_else_loops_functions.
En este directorio se encontraran funciones y/o programas utilziados para parender el uso de if/else, loops y funciones.
---
contenidos del directorio:
0-positive_or_negative.py - Imprime un numero al azar y dice si es positivo, negativo o cero.

1-last_digit.py - busca el ultimo digito de un numero al azar e imprime si es mas grande que 5, 0 o mas chico que 6 y no 0.

2-print_alphabet.py - imprime el abecedario en minuscula.

3-print_alphabt.py - imprime el abecedario sin la 'e' y la 'q'.

4-print_hexa.py - imprime del 0 al 98 y su equivalente en hexadecimal.

5-print_comb2.py - imprime del 0 al 99 utilziando un formato de dos cifras y al final de cada numero una ',' a excepcion del ultimo numero, sí el numero es de una cifra, rellena con 0 a la izquierda.
